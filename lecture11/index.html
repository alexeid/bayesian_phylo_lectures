Skyline Coalescent

An Introduction

What is Coalescent Theory?

  • A mathematical framework for studying the genealogical history of populations
  • Models the ancestry of individuals in a population by tracing them backward in time
  • Helps infer population parameters like mutation rate, population size, and gene flow

Basic Coalescent Model

  • Assumes constant population size
  • Each lineage has an equal chance of coalescing with any other lineage
  • Time between coalescent events follows an exponential distribution

Limitations of Basic Coalescent Model

  • Assumes constant population size
  • Does not account for changes in population size over time
  • Limited in capturing complex population dynamics

Skyline Coalescent Model

  • Extension of the basic coalescent model
  • Allows for variable population sizes over time
  • Estimates population size changes based on genealogies

How Does the Skyline Coalescent Work?

  • Divides the genealogy into intervals
  • Estimates population size for each interval
  • Creates a piecewise-constant function to describe population size changes

Bayesian Skyline Plot

  • A graphical representation of the Skyline Coalescent
  • Estimates population size changes over time
  • Displays credibility intervals for population size estimates

Applications of Skyline Coalescent

  • Understanding population dynamics
  • Inferring demographic history of species
  • Studying the spread of infectious diseases

Advantages and Limitations

  • Advantages:
    • Flexible model for variable population sizes
    • Useful for studying complex population dynamics
  • Limitations:
    • Computationally intensive
    • Requires high-quality genetic data

Conclusion

The Skyline Coalescent is a powerful tool for studying population dynamics and inferring demographic history. It extends the basic coalescent model by allowing for variable population sizes over time, providing a more realistic framework for analyzing complex population histories.
