Global alignment using affine gap scores

  • We can also easily modify the Needlemanâ€“Wunsch algorithm to compute optimal alignments with affine gap scoring.
  • Let $s(x_i , y_i)$ be the substitution score for characters $x_i$ and $y_i$.
  • If $x_1, x_2, \dots , x_n$ and $y_1, y_2, \dots, y_m$ are two strings then we can compute the best alignment $F(n,m)$ using:

    \begin{align*} F(i,j) = \max\left\{\begin{array}{l} F(i-1,j-1) + s(x_i,y_j)\\ F(i,k) + \gamma(j-k), \; k= 0,\dots,j-1\\ F(k,j) + \gamma(i-k), \; k= 0,\dots,i-1 \end{array}\right. \end{align*}
    $$F(k,0) = F(0,k) = \gamma(k) = -d - (k-1)e$$

    But, the running time is now cubic: $\Theta((n + m)^3)$ !
